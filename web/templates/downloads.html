Downloads

AEME encodes text in XML following the recommendations of the TEI Schema and its transcription guidelines. AEME XML code is open source so that the scholarly community can build upon the minimal requirements of AEME's schema. Source files are therefore made available on a version-controlled basis. AEME will publish download links with the following tags.

  • Release
  • Pre-Release
  • Non-Conformant
  • Backup

Files with the Release tag are official publications of AEME and are compliant with its minimal schema and available for rendering in the AEME platform. Files with the Pre-Release tag are generally development versions or contain encoding that cannot be rendered by the AEME platform. AEME may make these files available for downloading from a link on the AEME website but they are not official AEME publications and are not rendered on its viewing platform. Files with the Non-Conformant tag may be developed for use with other tools and may contain code that does not follow the AEME schema. AEME may choose to publish links to to this code for the benefit of the community. Files with the Backup tag are typically copies of files held in institutional repositories for preservation purposes, whereas those without this tag may be held in locations like GitHub.

Currently, AEME's files are all Pre-Release and may also be Non-Conformant. Although they can be viewed from AEME's GitHub development account, we are not yet ready to list download links on the AEME website.
